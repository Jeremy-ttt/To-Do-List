# To-Do-List

A simple to-do list flask appliaction,一个基于Flask的简洁的待办事项清单。

快来拥有属于自己的清单吧（

# 快速使用

### 安装环境

在python3环境下

`pip3 install Flask`

### 设置app端口

自行更改app.py中的`app.run("0.0.0.0",port=端口)`

若DNS解析直接域名+端口即可访问。

### 运行

命令行中

`cd 根目录`

`sudo python3 app.py`
